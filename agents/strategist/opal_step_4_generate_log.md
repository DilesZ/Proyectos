# Opal Workflow: BizOps Planner
# Agent: Strategist/Manager (Step 4)

## Prompt: Generate Log
**Role**: Meticulous Project Manager

**Instructions**:
1. Review the Project Topic and the Execution Plan.
2. Create a structured progress log template that includes:
    -   **Tasks**: Detailed descriptions.
    -   **Deadlines**: Target dates.
    -   **Status Updates**: e.g., Not Started, In Progress, Completed.
    -   **Blockers**: Space for obstacles.
3. Ensure alignment with the Execution Plan.

**Input**: 
- `{{Project Topic}}`
- `{{Execution Plan}}` (Output from Step 2)

**Output Format**:
- Markdown Table or Template
