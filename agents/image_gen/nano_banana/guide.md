## ‚ö° Pro Workflow: The "30-Image Datasheet"
To generate a consistent 30-image character sheet using Nano Banana Pro and external tools:

### Phase 1: The Master Reference (Gemini 3 Pro)
1.  **Tool**: Nano Banana Pro (Freepik/Gemini Advanced).
2.  **Prompt**: Use the "Character Anchor" prompt (see `prompts_datasheet.md`) to create a 3-view character sheet (Front, Side, 3/4).
3.  **Goal**: Lock in facial features, clothing, and style.

### Phase 2: Batch Generation (SeaArt / Freepik)
1.  **Tool**: SeaArt.ai (Batch Mode) or Freepik (Re-imagine).
2.  **Input**: Upload the "Master Reference" from Phase 1.
3.  **Control**: Use **ControlNet (Reference-Only)** or **InstantID** in SeaArt to keep the face 100% consistent.
4.  **Prompts**: Feed the list of 30 scenarios (e.g., "eating burger", "driving car", "holding laptop") generated by the Strategist Agent.

### Phase 3: Animation (Viyou.ai)
1.  **Tool**: Viyou.ai.
2.  **Action**: Upload selected datasheet images to generate short "Living Portraits" or "Dance" clips for social media engagement.

## üìù Best Practices (Freepik)
- **Prompts**: Be descriptive but concise. Use Freepik's style keywords (e.g., "digital art", "3d render", "studio lighting").
- **Reference Images**: Upload a reference of the character (e.g., Elena) to maintain consistency.
- **Workflow**:
    1. Define prompt in Strategist.
    2. Generate base image with Nano Banana.
    3. (Optional) Upscale or refine in Stable Diffusion if needed.
